# Projeto Inicial

Este e o primeiro projeto base hospedado na aba de "Projetos Academicos" do portfolio. Foi alocado para demonstrar uma organizacao fundamental dos diretorios de entrega de atividades.

## Objetivo

O foco principal na criacao desta secao e ter uma marcacao (HTML e CSS) simples e inteiramente isolada, sem dependencias externas, bibliotecas ou interatividade complexa, focando na pureza da linguagem visual.
