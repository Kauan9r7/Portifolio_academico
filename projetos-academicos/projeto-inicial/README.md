# Projeto Inicial

Este e o primeiro projeto base hospedado na aba de "Projetos Academicos" do portfolio. Foi alocado para demonstrar uma organizacao fundamental dos diretorios de entrega de atividades.

## Objetivo

O foco principal na criacao desta secao e ter uma marcacao (HTML e CSS) simples e inteiramente isolada, sem dependencias externas, bibliotecas ou interatividade complexa, focando na pureza da linguagem visual.

## Caracteristicas do Desenvolvimento

- Zero JavaScript: Foco estrutural sem scripts rodando em segundo plano. Segue as diretrizes consolidadas para leveza;
- Estilizacao Independente: A injecao de estilo e feita estritamente atraves do arquivo `style.css` anexado a este mesmo pacote, isolando regras do portfolio raiz;
- Navegacao Limpa: Facilmente interligado e escalavel conforme cresce o escopo de implementos de interface futuros.

## Reproducao

Trata-se de uma aplicacao estatica em nivel de visualizador simples. Voce so precisa acessar o diretorio deste pacote e abrir o arquivo `index.html` ali presente no seu navegador.
